		____________________
		| Utilizando o Vim |
		--------------------


Para editar arquivos no terminal do Git é preciso usar o vim (ou vi), digite vi ou vim e o nome do arquivo (ex $vi Teste.txt)

Caso não exista um arquivo com esse nome ele será criado, caso exista o editor VIM abrirá o arquivo

Para edita-lo aperte i e assim você poderá editar o documento.
Depois de terminado aperte esc e depois digite :wq (O w é utilizado para salvar as alterações e o q é utilizado para sair)


Para navegar no documento fora do modo de inserção utiliza-se:


	H - Esquerda

	L - Direita

	K- - Cima

	J - Baixo
